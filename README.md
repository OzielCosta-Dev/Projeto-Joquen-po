Jogo simples de **Pedra, Papel e Tesoura** feito com HTML, CSS e JavaScript.

O jogador escolhe uma das tres opcoes disponiveis e o computador faz uma escolha aleatoria. O resultado da rodada aparece na tela junto com a pontuacao do jogador e da maquina.

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## Como jogar

1. Abra o arquivo `index.html` no navegador.
2. Clique em uma das opcoes:
   - Pedra
   - Papel
   - Tesoura
3. Veja o resultado da rodada na tela.
4. A pontuacao e atualizada automaticamente.

## Regras do jogo

- Pedra ganha de Tesoura.
- Tesoura ganha de Papel.
- Papel ganha de Pedra.
- Opcoes iguais resultam em empate.
